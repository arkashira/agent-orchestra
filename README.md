# Agent Orchestra

A simple agent orchestra system that allows you to add, remove, and update agents, as well as get their status.

## Usage

To use the agent orchestra, simply run the `agent_orchestra.py` script with the desired arguments.

* `--add <agent_name>`: Add a new agent with the given name.
* `--remove <agent_name>`: Remove an existing agent with the given name.
* `--update <agent_name>,<status>`: Update the status of an existing agent with the given name.
* `--status <agent_name>`: Get the status of an existing agent with the given name.
* `--list`: List all existing agents.

## Testing

To run the tests, simply run `pytest` in the `tests` directory.
