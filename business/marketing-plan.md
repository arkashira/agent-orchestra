# Marketing Plan – agent-orchestra
**Product:** agent-orchestra  
**Repo:** https://github.com/arkashira/agent-orchestra  
**Hypothesis:** Lightweight, interactive multi‑agent orchestration for local CLI coding agents, no cloud billing, no over‑engineering.

| 1‑Line | Alternative 1 | Alternative 2 | Alternative 3 |

## 2. Ideal Customer Profile (ICP)

| Persona | Daily Job | Pain Point | Budget (USD) | Decision Triggers |
|---------|-----------|------------|--------------|-------------------|
| **Alex “CLI” Carter** | Front‑end engineer, 8‑hour coding days | Manages 3‑5 local LLM agents for code review, refactor, and documentation | $200–$400/month | Open‑source friendliness, zero cloud cost |
| **Maya “Ops” Singh** | DevOps engineer, 10‑hour shifts | Orchestrates CI pipelines with local agents for linting, security checks | $300–$600/month | Easy integration with existing scripts, CLI‑centric |
| **Eli “Fullstack” Ruiz** | Full‑stack developer, 9‑hour days | Needs to run multiple agents for testing, data generation, and debugging | $250–$500/month | Lightweight, no external dependencies, quick setup |
